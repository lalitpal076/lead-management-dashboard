# Lead Management Dashboard 
[![Live Demo](https://img.shields.io/badge/Live-Demo-blue)](https://lead-management-dashboard-3rr95xogc-lalit-s-team1.vercel.app/)



A simple Lead Management Dashboard built using **React** and **Bootstrap**.

## Features

* View and manage leads
* Clean user interface using Bootstrap
* Component-based architecture using React

## Technologies Used

* React
* Bootstrap


## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/lalitpal076/lead-management-dashboard.git
   ```

2. Navigate to the project folder:

   ```bash
   cd lead-management-dashboard
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

## Project Structure

* **React** for building the user interface
* **Bootstrap** for styling and responsive design
